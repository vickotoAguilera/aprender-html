# 01 - VISIÓN DEL PROYECTO

## Descripción General
Esta plataforma es un entorno educativo interactivo centrado en la construcción de un **Portfolio Personal**. El currículo ha sido rediseñado como un flujo de aprendizaje acumulativo, donde cada lección construye sobre la anterior, permitiendo al estudiante crear un proyecto profesional real mientras aprende HTML, CSS y JavaScript.

## Objetivos
- **Currículo Unificado**: Enseñar fundamentos web a través de un único hilo conductor (Portfolio).
- **Aprendizaje Acumulativo**: Uso de plantillas que mantienen el progreso del código entre lecciones.
- **Feedback Adaptativo**: Mentor IA con lógica robusta para guiar paso a paso.
- **Interfaz IDE Premium**: Experiencia inspirada en VS Code con vista previa en tiempo real.

## Alcance
- Editor de código basado en Monaco.
- Vista previa en vivo mediante iframe con RetroGrid.
- Sistema de misiones guiadas (Mentor IA).
- Biblioteca de lecciones académicas integradas.
